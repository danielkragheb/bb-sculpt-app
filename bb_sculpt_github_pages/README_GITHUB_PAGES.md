# BB Sculpt app — GitHub Pages free deployment

This package is prepared for **GitHub Pages**.

## What to upload
Upload **everything in this folder** into the root of a GitHub repository:
- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `icons/`
- `404.html`
- `.nojekyll`

## Fastest free deployment
1. Create a **new public GitHub repository**.
2. Upload all files from this folder to the **root** of that repository.
3. On GitHub, open the repository.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
6. Save.
7. Wait a minute or two. GitHub Pages will publish the site.
8. Your shareable link will look like:
   - `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## Notes
- This is a **front-end prototype**. It does not include live booking, payments, authentication, or a backend.
- It is mobile-first and opens well on iPhone Safari.
- Users can add it to the iPhone home screen using **Share → Add to Home Screen**.
